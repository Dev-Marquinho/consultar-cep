# Consulta de CEP

Projeto desenvolvido com JavaScript consumindo a API ViaCEP.

## Funcionalidades

- Buscar CEP
- Exibir logradouro
- Exibir bairro
- Exibir cidade
- Exibir estado

## Tecnologias

- HTML
- CSS
- JavaScript
- Fetch API
- ViaCEP API

## Aprendizados

- fetch()
- Promises
- response.json()
- Consumo de APIs
- Manipulação do DOM
